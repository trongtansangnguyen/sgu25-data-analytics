# VN30 Financial Dashboard

Ứng dụng Streamlit giúp trực quan hóa dữ liệu cho các cổ phiếu thuộc VN30.

## 1. Yêu cầu hệ thống

- Python >= 3.9 (khuyến khích bản 3.11)

- pip (trình quản lý gói Python)

- Các thư viện được liệt kê trong requirements.txt

## 2. Cài đặt

- Cài đặt các thư viện cần thiết: 
    > pip install -r requirements.txt

## 3. Chạy ứng dụng

Sau khi cài đặt xong, chạy lệnh sau trong thư mục labs/hw2:
    
> streamlit run vn30_dashboard.py 

Sau đó mở trình duyệt tại địa chỉ mà Streamlit hiển thị (http://localhost:8501)
